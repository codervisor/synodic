# Synodic Specs

Spec-driven development for Synodic. Each spec is a numbered directory containing a `README.md`.

## Status Legend

| Status | Meaning |
|--------|---------|
| `draft` | Idea captured, not yet designed |
| `planned` | Designed, ready for implementation |
| `in-progress` | Currently being implemented |
| `complete` | Implemented and verified |
| `deferred` | Valid but postponed until MVP validates |
| `archived` | Superseded or abandoned |
