# The Harness Governance Specification v0.1

This directory will contain the machine-readable specification for the Harness governance protocol, making it testable and adoptable by other projects without depending on Synodic's runtime.

## Planned Contents

| Path | Purpose |
|------|---------|
| `schema/governance-event.json` | JSON Schema for governance log entries |
| `schema/manifest.json` | JSON Schema for run manifests |
| `schema/rule.json` | JSON Schema for crystallized rules |
| `examples/factory-run.jsonl` | Example governance log from a factory run |
| `examples/crystallized-rule.json` | Example of a crystallized rule |

## Status

Stub — schemas will be added as the governance log format stabilizes after real-world usage.

See [HARNESS.md](../../HARNESS.md) for the current protocol specification.
